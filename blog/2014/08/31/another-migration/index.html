Another Migration

Aug 31st, 2014 12:34 am

It’s been a while since I posted anything on my blog. I have been using Wordpress for quite some time , Now I have decided to host my blog on Github via Octopress which works on Jekyll.

Some of the blogs I follow are hosted on Jekyll and I wanted to try it out. I struggled a bit with setting up the environment , but once it is set up , it works really well. One instant thing to notice is that it is faster as it doesn’t uses any kind of database. This is purely rendered through static pages.

Well , the set up isn’t difficult at all. If only I had run $ sudo apt-get install build-essential before. My installation was failing at $ gem install bundle.

This was the error I recieced:

This was fixed by $ sudo apt-get install build-essential

I had earlier set up my blog in Jekyll. But Octopress makes it easier managing Jekyll. For example running $ rake new_post['title'] auto generates the markdown file with appropriate names . Octopress has awesome plugins , for more features visit Octopress .

That’s it for now , I’ll probably transfer my Wordpress posts here if I get time!
